# Task T26: Viết E2E Testing (Frontend)

## 1. Mục tiêu

## 2. Phạm vi công việc
- [ ] Cài đặt Playwright.
- [ ] Viết test case: "User đăng ký -> đăng nhập -> tạo sản phẩm".

- [ ] Viết test case: "Admin đăng nhập -> xóa user".
